# Cloud Foundations

This repository documents my hands-on learning and practical labs covering:
- Linux fundamentals
- Networking concepts
- AWS core services

The goal is to build a strong foundation for cloud engineering and cloud security roles.

## Topics Covered
- Linux command-line usage and system basics
- Networking fundamentals (CIDR, DNS, routing)
- AWS EC2, IAM, S3, and VPC

## Approach
- Learn the concept
- Build it in practice
- Break it
- Fix it
- Document lessons learned

This repo is continuously updated as my skills progress.
